# AI-Agent
Building Apps using AI Agents
## project setup

1. Activate environment in powershell

```
.\.venv\Scripts\activate.ps1
```

2. install uv for fast package install
```
pip install uv
```

3. install packages
```
uv pip install <package_name>
```
4. After changing in .ts run this command 
```
 npx tsc hitl.ts --target ES2017 --lib ES2017,DOM
 ```
